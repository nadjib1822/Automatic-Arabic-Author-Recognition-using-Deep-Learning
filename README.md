# Automatic Arabic Author Recognition

This project presents an Automatic Author Recognition (AAR) system for Arabic texts using Natural Language Processing (NLP) and Deep Learning techniques. It aims to identify the author of a given Arabic text by learning each author's unique writing style.

## Technologies

* Python
* PyTorch
* Hugging Face Transformers
* Scikit-learn
* Pandas
* NumPy

## Models

* TF-IDF
* N-grams
* AraBERT
* AraELECTRA
* CAMeLBERT
* BiLSTM
* Transformer

## Dataset

An Arabic Islamic corpus containing texts from multiple authors.

## Results

Performance comparison of different models using Accuracy, Precision, Recall, and F1-score.
